# ChudWare

ChudWare is a Forge-based Minecraft client for **1.8.9**.
It is **free** and **fully open source**.

## Modules

- AimAssist
- ArrayList
- AutoBridge
- AutoClicker
- Chams
- ESP
- FullBright
- Reach
- Velocity


I am actively working on adding more modules :D

## Requirements

- Java 8
- Gradle wrapper (`gradlew`)

## Build

```bash
cd ChudWareSrc
./gradlew build
```
